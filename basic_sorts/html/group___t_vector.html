◆ TVector

struct TVector

Definice typu Vector.

Typ Vector spravuje dynamicky alokované pole prvků typu VectorElement a umožňuje s nimi pracovat pomocí definovaného API.

Invariant
Obecně platný a testovatelný předpoklad: (iValues == NULL && iSize == 0) nebo (iValues != NULL && iSize != 0).
