struct TQueue

Definice typu Queue.

Typ Queue obsahuje ukazatele na dynamicky alokované proměnné typu QueueNode, které představují uzly s hodnotami elementů uspořádanými do lineárního jednosměrně vázaného seznamu. Fronta umožňuje pracovat se svými elementy pomocí definovaného API.

Invariant
Obecně platný a testovatelný předpoklad: (iFront == NULL && iBack == NULL) nebo (iFront != NULL && iBack != NULL).
