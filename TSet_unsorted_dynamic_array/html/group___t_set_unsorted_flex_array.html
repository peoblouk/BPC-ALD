int set_flex_array_search ( const struct TSetUnsortedFlexArray * aFlexArray,
TSetElement aValue,
size_t * aPosPtr,
size_t aSize )

Funkce zjišťující, zda flexibilní pole obsahuje element o zadané hodnotě

Funkce (predikát) vracející bool hodnotu reprezentující test, zda flexibilní pole obsahuje zadanou hodnotu elementu.

Parametry
[in]aFlexArrayUkazatel na existující flexibilní pole
[in]aValueHodnota elementu hledaná v poli
[in,out]aPosPtrUkazatel na proměnnou určenou pro uložení pozice na hledaný element v poli (hodnota pozice je do paměti zapsána vždy tj. bez ohledu na výsledek hledání)
[in]aSizeAktuální počet elementů ve flexibilním poli
Vracené hodnoty
-2Pokud má pole nulovou délku, (*aPosPtr = 0)
-1Pokud element nebyl v poli nalezen a hledaná hodnota elementu je menší, než hodnota na poslední platné pozici *aPosPtr
0Pokud element byl v poli nalezen a hledaná hodnota elementu je na pozici *aPosPtr
+1Pokud element nebyl v poli nalezen a hledaná hodnota elementu je větší, než hodnota na poslední platné pozici *aPosPtr
Poznámka
Pokud element s danou hodnotou nebyl nalezen, obsahuje *aPosPtr poslední platnou pozici, kde bylo hledání ukončeno.
Upozornění
Funkce ověřuje platnost obou ukazatelů pouze při překladu v režimu Debug, kdy pomocí assert hlásí běhovou chybu!
