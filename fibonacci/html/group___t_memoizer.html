Inicializace memoizeru, asociace s memoizovanou funkcí, alokace vnitřního pole pro pár hodnot vstup a výstup memoizované funkce.

Dynamicky alokuje paměť pro pole o zadaném počtu prvků, provede asociaci s memoizovanou funkcí, inicializuje všechny složky na poč. hodnoty.

true pokud byl memoizer úspěšně inicializován a alokováno vnitřní pole

◆ memoizer_eval()

static TMemoizedFuncOutput memoizer_eval ( struct TMemoizer * aM,
TMemoizedFuncInput aInput )
inlinestatic

Volání memoizeru zajišťující získání výsledku asociované funkce pro zadaný parametr.

Prohledá archiv zapamatovaných párů vstup, výstup a vrací uloženou návratovou hodnotu asociované funkce pro zadanou hodnotu parametru. Pokud návratová hodnota funkce pro zadanou vstupní hodnotu nebyla nalezena (není archivována), určí výsledek pomocí volání asociované funkce a tuto hodnotu si archivuje. Pokud je archiv již plný, přepíše nejstarší archivovaný pár.

Parametry
[in,out]aMUkazatel na existující memoizer
[in]aInputHodnota požadovaného vstupního parametru asociované funkce
Návratová hodnota
Návratová hodnota asociované funkce pro zadaný vstupní parametr

◆ memoizer_clear()

static bool memoizer_clear ( struct TMemoizer * aM)
inlinestatic

Smazání archivovaných hodnot v memoizeru, vynulování statistik o počty volání memoizeru.

Smaže archivované páry pro vstup a návratovou hodnotu memoizované funkce, vynuluje statistiky o počty volání memoizeru. Ponechává asociaci s memoizovanou funkcí a také ponechá alokovanou paměť pole hodnot typu MemoizerItem tak, aby bylo možné memoizer znovu používat.

Parametry
[in,out]aMUkazatel na existující memoizer jehož archiv a statistiky chceme smazat
Návratová hodnota
true pokud byl ukazatel na memoizer platný a zapamatované hodnoty a statistiky byly vymazány
