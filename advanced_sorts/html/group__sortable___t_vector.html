bool vector_is_ascendantly_sorted ( const struct TVector * aUnsortedVector,
const struct TVector * aSortedVector )

Ověření korektního setřídění vektoru.

Ověřuje správnost vzestupného setřídění zadaného vektoru. Také ověřuje, zda setříděný vektor obsahuje shodné hodnoty elementů jako nesetříděný vektor.

Parametry
[in]aUnsortedVectorUkazatel na původní (nesetříděný) vektor
[in]aSortedVectorUkazatel na nový (setříděný) vektor
Návratová hodnota
true pokud nový vektor aSortedVector obsahuje všechny hodnoty elementů jako původního vektor aUnsortedVector a tyto elementy jsou vzestupně setříděny
