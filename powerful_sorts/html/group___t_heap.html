Definice typu HeapIterator.

Invariant
Obecně platný a testovatelný předpoklad: (iVector->iValues == NULL && iVector->iSize == 0 && iOriginalSize == 0) nebo (iVector->iValues != NULL && iVector->iSize != 0 && iOriginalSize >= iVector->iSize).
